Split Indications and Backfill

将 Excel 中以分号分隔的多适应症记录，拆分为“一行一个适应症”的可追溯底稿，并完整回填原始字段。

What it does

仅按已确认的分隔符拆分适应症；

保留原始记录和全部原始字段；

为每条拆分记录保留唯一标识和药品/实体名称；

通过 XLOOKUP 回填完整原始信息；

生成可供后续疾病 Mapping 使用的标准化工作簿。

Output

输出工作簿包含：

原始底稿：保持不变，作为唯一事实来源；

拆分结果：一行一个适应症，便于快速核阅；

更新版拆分底稿：保留完整字段，供后续 Mapping 使用。

Key safeguards

不修改、覆盖或删除原始输入文件；

唯一主键不为空且不重复，否则暂停；

适应症空值记录不删除；

不擅自拆分逗号、顿号、斜杠或括号内容；

正式批量输出前，先展示样例并获得人工确认。

Typical use case

适用于药品、分子或资产池中一条记录包含多个获批适应症，且后续需要逐项开展疾病 Mapping、治疗领域筛选或分子池分析的场景。
